# Telecom Customer Churn Prediction (82% Accuracy)

## Project Summary  
Customer churn is a critical challenge in the telecom industry. This project builds a predictive system to identify customers likely to leave, achieving **82% accuracy** using an **ensemble machine learning approach**.  

The solution not only predicts churn but also translates results into **actionable business insights** through an interactive dashboard.

## Business Problem  
Acquiring new customers is significantly more expensive than retaining existing ones. The goal is to:  
- Detect at-risk customers early  
- Understand key churn drivers  
- Support targeted retention strategies  

## Solution Approach  
- Built a **classification model** using ensemble techniques  
- Combined multiple algorithms to improve robustness and accuracy  
- Transformed predictions into insights via a **custom dashboard**

## Tech Stack  
- **Programming**: Python  
- **Data Analysis**: Pandas, NumPy  
- **Modeling**: Scikit-learn (Ensemble Models)  
- **Visualization**: Matplotlib, Seaborn  
- **Dashboard**: Streamlit  

## Dataset Features  
The dataset includes:  
-  Customer demographics (gender, tenure, etc.)  
-  Account details (contract type, billing method)  
-  Service usage (internet, phone services)  
-  Charges (monthly & total charges)  

## Model Performance  
- Accuracy: **82%**  
- Improved prediction stability using ensemble learning  
- Identified high-impact features influencing churn  

## Key Insights  
- Customers with **month-to-month contracts** show higher churn  
- Higher **monthly charges** correlate with increased churn risk  
- Long-term customers are less likely to churn  

## Dashboard Highlights  
- Churn vs Non-Churn distribution  
- Customer risk segmentation  
- Feature importance visualization  
- Interactive filtering for business analysis
